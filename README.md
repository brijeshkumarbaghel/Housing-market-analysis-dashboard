# Housing Market Analysis Dashboard | Power BI + SQL + BigQuery

## Project Overview
This project is an end-to-end Housing Market Analysis Dashboard developed using Google BigQuery, SQL, Power BI, Power Query, and DAX to analyze housing sales trends, regional performance, pricing behavior, and sales growth patterns.

The dashboard provides actionable insights into:
- Year-over-Year (YOY) Sales Growth
- Regional Sales Performance
- Offer Price vs Purchase Price Analysis
- Median Sales Price Change
- Last 12 Months Sales Trend
- House Type Analysis
- Key Influencers affecting Purchase Price

This project demonstrates practical skills in:
- Data Cleaning & Transformation
- SQL-based Data Analysis
- Data Modeling
- Advanced DAX Calculations
- Interactive Dashboard Development
- Business Insight Generation

## Project Workflow

### 1. Cloud Data Integration
- Created a Google Cloud account and uploaded housing market data into Google BigQuery
- Connected BigQuery with Power BI for cloud-based analytics

### 2. SQL Data Analysis
- Performed data exploration, filtering, aggregations, and transformations using SQL in BigQuery

### 3. Data Cleaning using Power Query
- Cleaned and transformed raw housing data
- Removed inconsistencies and prepared the dataset for reporting

### 4. Advanced DAX Calculations
Created multiple business KPIs and measures using DAX functions such as:
- CALCULATE
- MEDIANX
- DATESINPERIOD
- TOTALYTD
- DISTINCTCOUNT
- ALLEXCEPT
- DIVIDE
- YEAR
- QUARTER
- IF
- BLANK

Key Measures:
- YOY Sales Growth
- Last 12 Month Sales
- Median Sales Price Change
- Sales by Region
- Units Sold in Latest Year & Quarter
- Offer to SQM Ratio
- Total YTD Sales

## Key Insights
- Zealand region generated the highest housing sales revenue
- Auction sales showed positive YOY sales growth
- Properties aged between 3–15 years contributed to higher purchase prices
- Apartment and Villa house types recorded higher SQM prices
- Regular sales had the highest Offer-to-SQM ratio
